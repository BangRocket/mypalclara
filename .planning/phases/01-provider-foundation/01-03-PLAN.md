<task type="auto">
  <name>Task 2: Integrate ProviderManager into gateway startup</name>
  <files>gateway/main.py</files>
  <action>
Modify gateway/main.py to initialize and manage providers during startup/shutdown.

**Add imports:**
```python
from gateway.providers import get_provider_manager, DiscordProvider
```

**Add command-line argument:**
```python
parser.add_argument(
    "--enable-discord",
    action="store_true",
    default=os.getenv("CLARA_GATEWAY_DISCORD", "false").lower() == "true",
    help="Enable Discord provider (default: $CLARA_GATEWAY_DISCORD or false)",
)
```

**In main() function, after hooks and scheduler init:**

1. Initialize ProviderManager:
```python
provider_manager = get_provider_manager()
```

2. Register Discord provider if enabled:
```python
if args.enable_discord:
    discord_provider = DiscordProvider()
    provider_manager.register(discord_provider)
    logger.info("Discord provider registered")
```

3. Start providers AFTER server starts (before the "ready" log message):
```python
# Start registered providers
if provider_manager.providers:
    logger.info(f"Starting {len(provider_manager.providers)} provider(s)...")
    await provider_manager.start_all()
    logger.info("All providers started")
```

4. In shutdown sequence, BEFORE server.stop():
```python
# Stop providers first (graceful shutdown)
if provider_manager.providers:
    logger.info("Stopping providers...")
    await provider_manager.stop_all()
    logger.info("All providers stopped")
```

**Update parse_args():**
- Add enable_discord to returned args
- Pass it through to main()

**Update main() signature:**
```python
async def main(host: str, port: int, hooks_dir: str, scheduler_dir: str, enable_discord: bool) -> None:
```

**Important:** Discord provider is OPTIONAL in Phase 1. Default is disabled.
