```python
"""Discord provider for Clara Gateway.

Implements the Strangler Fig pattern: wraps the existing ClaraDiscordBot
without rewriting its core logic. This allows gradual migration while
preserving all proven Discord bot functionality.
"""

from __future__ import annotations

import asyncio
import os
from typing import TYPE_CHECKING, Any

from clara_core.platform import PlatformMessage
from config.logging import get_logger
from gateway.providers.base import Provider

if TYPE_CHECKING:
    from discord import Message as DiscordMessage

logger = get_logger("gateway.providers.discord")
```

**Class structure:**

1. `__init__(self)`:
   - `self._bot: ClaraDiscordBot | None = None`
   - `self._bot_task: asyncio.Task | None = None`
   - `self._running = False`

2. `name` property returns `"discord"`

3. `running` property returns `self._running`

4. `async def start(self)`:
   - Import ClaraDiscordBot here (avoids circular imports)
   - Get token from `os.getenv("DISCORD_BOT_TOKEN")`
   - Raise RuntimeError if no token
   - Create bot instance: `self._bot = ClaraDiscordBot()`
   - Set `self._bot._provider_mode = True`
   - Start bot in background task: `self._bot_task = asyncio.create_task(self._bot.start_for_provider(token))`
   - Wait for bot ready with timeout: poll `self._bot.is_ready_for_provider()` up to 30s
   - Set `self._running = True`
   - Log success

5. `async def stop(self)`:
   - Set `self._running = False`
   - If `self._bot`: call `await self._bot.stop_for_provider()`
   - If `self._bot_task`: cancel and await it (catch CancelledError)
   - Reset `self._bot` and `self._bot_task` to None
   - Log shutdown

6. `def normalize_message(self, discord_msg: DiscordMessage) -> PlatformMessage`:
   - Follow pattern from adapters/discord/adapter.py
   - Convert discord.Message to PlatformMessage
   - Include metadata: is_dm, guild_id, guild_name, message_id, channel_name
   - Keep _discord_message in metadata for delegation

7. `async def send_response(self, context: dict, content: str, files: list[str] | None = None)`:
   - Extract channel from context (context["channel"])
   - Call `await context["channel"].send(content, files=files or [])`
   - Handle file conversion if needed (paths to discord.File)

8. `@property def bot(self) -> ClaraDiscordBot | None`:
   - Return `self._bot` for direct access when needed

**Important:** Use TYPE_CHECKING for discord imports to avoid import errors when discord.py is not available.
